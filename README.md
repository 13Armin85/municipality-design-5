# پرتال خدمات شهری شهرداری

این پروژه یک رابط کاربری فارسی و راست‌به‌چپ برای پرتال خدمات شهری شهرداری است که با استفاده از React و Vite توسعه داده شده است. کاربران از طریق این سامانه می‌توانند خدمات مختلف شهری را مشاهده، ثبت و پیگیری کنند؛ از جمله مشاهده املاک، ثبت درخواست، پرداخت و استعلام عوارض، پیگیری پرونده‌ها و استفاده از پنل مدیریتی.

---

## امکانات سامانه

* مشاهده و مدیریت املاک
* ثبت و پیگیری درخواست‌های شهری
* استعلام عقب‌نشینی ملک
* مشاهده و پرداخت عوارض نوسازی
* مدیریت عوارض صنفی
* مشاهده اخبار و اطلاعیه‌ها
* پروفایل کاربری
* پنل مدیریت
* پشتیبانی کامل از زبان فارسی و چیدمان RTL
* ذخیره وضعیت ورود و تنظیمات کاربر در LocalStorage
* محافظت از مسیرهای حساس با ProtectedRoute

# نصب و اجرای پروژه

ابتدا وابستگی‌ها را نصب کنید:

```bash
npm install
```

اجرای پروژه در حالت توسعه:

```bash
npm run dev
```

پس از اجرا، پروژه معمولاً در آدرس زیر در دسترس خواهد بود:

```text
http://localhost:5173
```

ساخت نسخه Production:

```bash
npm run build
```

# ساختار کلی پروژه

```text
src/
├── app/
│   ├── components/      # کامپوننت‌های عمومی و UI
│   ├── data/            # داده‌های نمونه و helperها
│   ├── hooks/           # هوک‌های اشتراکی
│   ├── pages/           # صفحات اصلی برنامه
│   └── App.tsx          # مسیرها و ساختار اصلی برنامه
│
├── styles/              # فایل‌های استایل، فونت و تم
└── main.tsx             # نقطه ورود برنامه
```

# اتصال به بک‌اند

در محیط توسعه، تمامی درخواست‌های `/api` از طریق Vite به بک‌اند پراکسی می‌شوند.

تنظیمات مربوط به پراکسی در فایل زیر قرار دارد:

```ts
vite.config.ts
```

نمونه تنظیمات:

```ts
target: "http://192.168.10.3:6300"
```

در صورتی که بک‌اند روی آدرس دیگری اجرا می‌شود، کافی است مقدار `target` را تغییر دهید.

# نکات مهم

* پروژه برای نمایش کامل داده‌ها نیازمند اتصال به بک‌اند است.
* اطلاعاتی مانند توکن ورود، نوع کاربر و تنظیمات تم در `localStorage` ذخیره می‌شوند.
* مسیرهای خدماتی مهم توسط `ProtectedRoute` محافظت شده‌اند و قبل از دسترسی نیاز به ورود کاربر دارند.
* برای اضافه کردن صفحه جدید:

  1. فایل صفحه را در `src/app/pages` ایجاد کنید.
  2. مسیر مربوطه را در `src/app/App.tsx` ثبت کنید.

---

# توسعه و شخصی‌سازی

برای توسعه قابلیت‌های جدید می‌توانید:

* کامپوننت‌های جدید را در پوشه `components` اضافه کنید.
* صفحات جدید را در `pages` ایجاد کنید.
* APIهای جدید را از طریق Proxy یا سرویس‌های مجزا متصل کنید.
* تم و استایل پروژه را از طریق Tailwind و فایل‌های موجود در `styles` شخصی‌سازی کنید.
